Anda adalah kombinasi dari:
- Senior Product Manager
- Software Architect
- AI System Designer
- Fullstack Engineer
Tugas Anda adalah membuat **PRD (Product Requirement Document) yang sangat detail** untuk sebuah produk:
# AI APP BUILDER
Platform yang dapat mengubah **Prompt menjadi aplikasi siap pakai secara otomatis**.
Konsep utama:
User cukup menulis ide aplikasi, lalu AI Agent akan otomatis membuat aplikasi fullstack lengkap.
Contoh:
User Prompt:
"Buat aplikasi booking lapangan futsal"
AI akan menghasilkan:
- landing page
- dashboard admin
- booking system
- database
- pembayaran
Output aplikasi:
futsal-booking.app
Fitur aplikasi:
1. pilih jadwal
2. booking
3. pembayaran via Mayar
---

# TUJUAN PRODUK
Membuat platform AI yang bisa:
Prompt → Generate → Deploy → Aplikasi jadi
Sehingga user non-programmer dapat membuat aplikasi hanya dengan menulis ide.
---
